# rabai

Half-baked wrapper on the `yabai` CLI with commands I like. 

```txt
Usage: rabai <COMMAND>

Commands:
  next               Focus on the next window (cycles)
  previous           Focus on the previous window (cycles)
  swap               Swap the current window with the next window (cycles)
  config             Set configuration options
  resize             Resize the window's right border to the left
  toggle-fullscreen  Toggle fullscreen
  help               Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
```
